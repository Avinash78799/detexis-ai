Detailed Workflow Steps

1. Data Ingestion

	Input: Kaggle Cybersecurity Intrusion Detection Dataset (CSV). https://www.kaggle.com/datasets/dnkumars/cybersecurity-intrusion-detection-dataset
	Load into Pandas DataFrame.

2. Preprocessing
	
	Handle missing values (drop or impute).
	Label encode categorical variables.
	Standardize numeric features.
	

3. Feature Selection and Splitting the Dataset

4. Model Training

	Train multiple algorithms:
		Random Forest (RF)
		Support Vector Machine (SVM)
		XGBoost (XGB)
		Artificial Neural Network (ANN)
		Logistic Regression (LR)
		Ensemble Model

5. Evaluation

	Evaluate on test set using:
		Accuracy, Precision, Recall, F1-score, ROC-AUC.
		Confusion matrix to see false positives & negatives.
	XAI

6. Save Artifacts

	Save best performing model using joblib.
	Save StandardScaler & LabelEncoder for Flask preprocessing.

7. Flask Web App

	Frontend (HTML):
		Input form for network traffic features.
	Backend (Flask):
		Load saved model, scaler, encoder.
		Preprocess incoming features.
		Predict intrusion type.
	Return result to user.
